Un paquete de iconos bifurcado de ICEcons con el objetivo de proveer iconos principalmente para F-Droid y otras aplicaciones FOSS. Está diseñado para verse limpio y sencillo, con solo iconos blancos con transparencia. Frost incluye soporte para Trebuchet, KISS, Nova, Apex, Holo, ZenUI, ADW, y muchos más lanzadores. Hay más de <b>1000 iconos</b>, así como algunos <b>fondos de pantalla 4K</b> incluidos.

Enviar solicitudes de iconos puede ser útil, pero mantener el proyecto requiere mucho tiempo y esfuerzo, así que tenga paciencia con los nuevos iconos. La mayoría de los iconos son el resultado de un pull request de alguien que deseaba agregar soporte para una aplicación específica. Las contribuciones para nuevos iconos son siempre bienvenidas y no se necesita experiencia en programación.
